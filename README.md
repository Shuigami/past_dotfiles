Directories
---
The files `.vimrc` and `.Xresources` must be in your home folder, the folder named `Wallpaper` should be in your `Pictures` folder the rest should be in your `.config` folder as followed :
```
~/.vimrc
  .Xresources
  Pictures/
      Wallpaper/
  .config/
      BetterDiscord/
      i3/
      polybar/
      rofi/
```
> **WARNING**: Be careful, or else!
